# Disney+ Clone - Proyecto Base

Este repositorio contiene una versión simplificada y limpia de la interfaz de Disney+, enfocada en el centrado de contenedores y una estructura de navegación funcional.

## Características
- Contenedor principal centrado de 1300px.
- Barra de navegación con Flexbox.
- Sección Hero con fondo de imagen y gradiente.
- Sistema de tarjetas responsivo mediante Flexbox.

## Requisitos
- Navegador web moderno.
- Imágenes en la carpeta `./images/`.
- Mucha creatividad.

## Estructura de Archivos
- `index.html`: Estructura semántica del sitio.
- `style.css`: Estilos básicos y diseño de layouts.

## Instalación
1. Clona o descarga este repositorio.
2. Asegúrate de tener la carpeta `images` con los recursos necesarios.
3. Abre `index.html` en tu navegador.

## Notas de Diseño
Se ha utilizado `margin: 0 auto` en la clase `.container` para asegurar que todo el contenido se mantenga centrado en pantallas grandes, mientras que el `body` utiliza un color de fondo gris claro para resaltar el área de contenido blanca.
